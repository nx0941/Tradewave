# Tradewave Ingredients Premium Website

A luxury global B2B website for Tradewave Ingredients Pvt. Ltd.

## Pages
Home, About, Products, six product pages, Custom Solutions, Quality, Global Markets, Vision, Contact.

## Publish on GitHub Pages
1. Upload all files to a GitHub repository.
2. Go to Settings > Pages.
3. Select branch `main` and root folder.
4. Your website will be published.

## Important
The contact form is visual only. Connect it to Formspree, Netlify Forms, EmailJS, or a backend before going live.
